学习编程的第一步，永远是把开发环境配置好。这就像你想学做饭，总得先把锅碗瓢盆买齐。

对于 Java 来说，你需要安装两个东西：
1. **JDK (Java Development Kit)**：这是 Java 语言的“引擎”。没有它，你的电脑根本看不懂 Java 代码。
2. **IDEA (IntelliJ IDEA)**：这是我们要用的“超级记事本”。它能帮我们自动补全代码、检查拼写错误，非常好用。

准备好了吗？跟着步骤一步一步来，保证不出错！

---

## 第一步：下载并安装 JDK

### 1. 去哪下载？
打开浏览器，进入 Oracle 官方下载页面（或者直接搜索 `JDK 17 download`）：
👉 [点击前往 Oracle JDK 下载页](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

> **💡 贴心提示**：为什么选 JDK 17？因为它是目前企业里用得最多、最稳定的“长期支持版(LTS)”。不要去追求最新版本，稳定才是第一位。

### 2. 下载哪个文件？
如果你用的是 **Windows** 电脑：
- 找到 `Windows x64 Installer`。
- 点击后面以 `.exe` 结尾的链接下载。

### 3. 开始安装
双击下载好的 `.exe` 文件，一路点击“下一步 (Next)”即可。
**重要提醒**：请务必记住安装路径！默认一般是 `C:\Program Files\Java\jdk-17`。这个路径我们在下一步马上要用到！

---

## 第二步：配置环境变量（最容易出错的一步！）

装好了引擎，我们还得告诉 Windows 系统引擎装在哪里了，这就是“配置环境变量”。

### 1. 打开环境变量设置
- 在 Windows 桌面底部任务栏的**搜索框**里，输入 `环境变量`。
- 点击搜索结果中的 **编辑系统环境变量**。
- 在弹出的窗口右下角，点击 **环境变量(N)...** 按钮。

### 2. 新建 JAVA_HOME
- 在下方**“系统变量(S)”**那一栏，点击 **新建(W)...**。
- 变量名输入：`JAVA_HOME` （必须全部大写，中间是下划线）。
- 变量值输入：刚才 JDK 的安装路径（例如 `C:\Program Files\Java\jdk-17`）。
- 点击“确定”。

### 3. 修改 Path 变量
- 依然在下方**“系统变量(S)”**那一栏，找到名叫 `Path` 的变量，双击它。
- 在弹出的新窗口中，点击右侧的 **新建(N)**。
- 输入这串神奇的代码：`%JAVA_HOME%\bin`
- 点击“确定”，把所有刚才打开的窗口都点“确定”关掉。

### 4. 验证是否成功
- 按下键盘上的 `Win + R` 键，输入 `cmd`，按回车，打开黑色的命令提示符窗口。
- 在里面输入：`java -version` 然后按回车。
- 如果你能看到类似 `java version "17.0.x"` 的文字，**恭喜你，Java 引擎安装成功！** 🎉

---

## 第三步：安装写代码的神器 —— IDEA

写代码用普通的记事本太折磨人了，全人类都在用 IntelliJ IDEA 这款软件写 Java。

1. **下载**：前往 [JetBrains 官网](https://www.jetbrains.com/idea/download/)。
2. **选择版本**：网页往下稍微拉一点，找到 **IntelliJ IDEA Community Edition (社区版)**。这个版本是完全免费的，对我们初学者来说绝对够用了！点击 Download。
3. **安装**：双击下载好的文件，依然是一路“Next”到底。如果中间有问你要不要创建桌面快捷方式（Create Desktop Shortcut），记得打个勾。

## 总结

到这里，做饭的“锅（JDK）”和“案板（IDEA）”都已经准备就绪了！

喝口水休息一下。在下一篇文章中，我们将打开 IDEA，敲下你人生中的第一行 Java 代码，并且逐字逐句地教你这行代码到底是什么意思。

**遇到问题了吗？**
> **防坑指南**：
> 1. 如果你在黑窗口里输入 `java -version` 提示“不是内部或外部命令”，那 100% 是你在**第二步配置环境变量**时把路径或者字母打错了，请仔细检查。
> 2. `JAVA_HOME` 的路径里**千万不要**带上 `\bin`，`\bin` 只能写在 `Path` 里面。
