在上一篇文章中，我们已经把做饭的“锅（JDK）”和“案板（IDEA）”准备好了。
今天，我们要正式下厨，炒一盘编程界最著名的菜——**Hello World**（你好，世界）。

别怕，即使你对电脑一窍不通，跟着步骤来，你也一定能成功让电脑对你开口说话！

---

## 第一步：在 IDEA 中新建项目

打开你安装好的 IDEA，跟着做：
1. 点击屏幕中央大大的 **New Project (新建项目)** 按钮。
2. 在左边选择 **Java**。
3. 在右边的 **Name (项目名称)** 里填入：`HelloJava`（注意没有空格，两个词首字母大写）。
4. 下面的 **Location (位置)** 是代码存放的地方，最好不要放在 C 盘，找个你记得住的文件夹。
5. **JDK** 选择我们上节课安装好的版本（比如 17）。
6. 其他什么都不用管，直接点右下角的 **Create (创建)**。

创建成功后，左边会出现一个文件夹目录。你要找到 `src` 文件夹（也就是 source 源码的意思），右键点击它：
选择 **New -> Java Class (新建 Java 类)**。
在弹出的框里输入 `Main`，按回车。

---

## 第二步：手写代码（不要复制！）

现在，屏幕右边应该出现了一个空白的代码文件。**请你亲自在键盘上，把下面这几行字原封不动地敲上去**（注意大小写和标点符号必须是英文）：

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```

敲完之后，点击第二行旁边出现的**绿色三角形播放按钮**（Run 'Main.main()'）。
稍等两秒钟，你会看到屏幕下方弹出一个窗口，里面打印出了：
`Hello World!`

**恭喜你！你已经成为一名写过 Java 代码的程序员了！** 🎉

## 第三步：这段代码到底是什么意思？

很多人学 Java，老师都会说：“这几行代码是固定格式，先死记硬背下来以后慢慢懂”。
**但我不这么认为。** 我们不能对写出来的东西一头雾水。让我们逐字逐句把这层窗户纸捅破！

### `public class Main` 
- **`public` (公开的)**：意思是这个东西是公开可见的，谁都可以来看。
- **`class` (类)**：Java 中最核心的概念。你可以把它理解为图纸、模板，或者一个“大房间”。Java 规定：**任何代码，都必须写在某个“房间（类）”里面。**
- **`Main`**：这是你给这个房间取的名字。注意：这里写的名字，必须和左侧文件的名字 `Main.java` 一模一样！
- **`{ }` (大括号)**：大括号用来划定地盘。左括号 `{` 代表房间开始了，右括号 `}` 代表房间结束了。

### `public static void main(String[] args)`
这一长串东西，叫做**方法（Method）**。你可以把它理解为这个房间里的一个“任务执行器”。
**在 Java 中，名叫 `main` 的方法非常特殊，它是整个程序的入口。** 电脑运行程序时，它只认准 `main`，会一头扎进这里面开始执行代码。

我们来看看前面这一串修饰词是什么意思：
- **`public`**：同样是公开的，说明电脑可以直接跑过来执行这个主任务。
- **`static` (静态的)**：意思是这个任务提前就准备好了，不需要什么前提条件就能直接执行。
- **`void` (空的/没有返回值)**：意思是这个任务执行完就完了，不需要向上级汇报什么成果。
- **`String[] args`**：这是个参数。就像你去奶茶店点单，你可以说“要冰的”、“要全糖”，这就是传进去的参数。这里暂且不用管它。

### `System.out.println("Hello World!");`
这是真正干活的那一行代码！
- **`System`**：系统。
- **`out`**：输出。
- **`println`**：打印并换行（print line 的缩写）。
所以连起来就是：“呼叫系统，把括号里面的内容，给我输出并打印到屏幕上！”

最后，千万不要忘了这行代码结尾的**分号（`;`）**！
在 Java 里，分号就像是我们写作文时的句号。没有它，电脑就不知道这句话在哪结束。

## 总结避坑指南

> **常见报错自查表**：
> 1. **代码下面出现红色的波浪线？** 检查是不是把 `System` 的大写 S 写成了小写 `s`。Java 严格区分大小写！
> 2. **运行报错 `Cannot resolve symbol`？** 检查是不是把英文的双引号 `""` 打成了中文的 `“”`。
> 3. **报错 `class Main is public, should be declared in a file named Main.java`？** 你的代码里写的类名，和左侧实际的文件名没有完全一致。
