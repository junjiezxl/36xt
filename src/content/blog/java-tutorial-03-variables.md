在上一节课中，我们成功让电脑说出了“Hello World!”。
但这有个问题：电脑只是像个复读机一样，把我们写死在代码里的字念出来，它自己没有任何**“记忆能力”**。

如果我们要写一个计算器，或者记录游戏角色的血量，我们必须让电脑帮我们“记住”一些随时会变化的数据。

用来记住这些数据的容器，在编程中就叫做**变量（Variable）**。

---

## 1. 什么是变量？

你可以把变量想象成一个**带标签的杯子**。
当我们需要记住角色的血量是 100 时，我们就在桌上放一个杯子，给杯子贴上标签（名字叫 `hp`），然后往杯子里装入 `100` 这个数字。

以后我们要扣血，就把杯子里的数字倒出来，减去一点，再装回去。

**但是在 Java 里有一个极其严格的规定**：
在拿杯子之前，**你必须先声明这个杯子是用来装什么的**。装水的杯子绝对不能用来装沙子！这就是 Java 所谓的“强类型语言”特性。

## 2. Java 里的四种常用“杯子”（数据类型）

Java 里有很多种杯子，但作为初学者，你只需要死死记住以下这 4 种最常用的：

### ① `int`（整数型）
最常用的数字杯子，专门用来装**没有小数点的整数**。比如年龄、血量、点赞数。

```java
// 语法：数据类型 变量名 = 存入的值;
int age = 18;
int hp = 100;
```

### ② `double`（双精度浮点型）
如果你要装**带有小数点的数字**，比如身高、体重、商品价格，就必须用 `double`。

```java
double price = 9.9;
double weight = 65.5;
```

### ③ `boolean`（布尔型）
这个杯子非常特殊，它只能装两个词中的一个：**`true`（真/是）** 或者 **`false`（假/否）**。
专门用来做判断题，比如“是否已经成年”、“是否会员”。

```java
boolean isAdult = true;
boolean isVip = false;
```

### ④ `String`（字符串）
用来装**文字、句子**。
**注意！** 所有的 `String` 内容，必须用英文的**双引号 `""`** 包裹起来！并且 `String` 的首字母 `S` 必须大写！

```java
String myName = "张三";
String message = "你好，Java！";
```

## 3. 实战：写一个“自我介绍”小程序

打开上一节课的 IDEA，把 `main` 方法大括号 `{ }` 里的代码删掉，跟着我敲下这段代码：

```java
public class Main {
    public static void main(String[] args) {
        // 1. 准备好杯子（声明变量）
        String name = "阿陈";
        int age = 22;
        double height = 1.75;
        boolean isLearningJava = true;

        // 2. 把杯子里的东西和别的文字拼接起来，打印出去
        // 在 Java 里，用加号 + 可以把文字和变量连在一起
        System.out.println("大家好，我的名字是：" + name);
        System.out.println("我今年 " + age + " 岁");
        System.out.println("我的身高是 " + height + " 米");
        System.out.println("我正在学习Java吗？" + isLearningJava);
    }
}
```

敲完后，点击绿色的三角形运行。
你会看到屏幕下方完美输出了这四行自我介绍！

## 总结避坑指南

> **常见报错自查表**：
> 1. **代码报红：`int age = "18";`** 
>    *原因：杯子用错了！`int` 只能装数字，你加了双引号 `"18"` 它就变成文字（String）了。*
> 2. **代码报红：`string name = "张三";`**
>    *原因：`String` 的首字母忘了大写！在 Java 里，首字母大小写非常严格。*
> 3. **打印时出问题：`System.out.println("年龄是" + Age);`**
>    *原因：上面定义的是 `age`，下面用的是 `Age`。大小写不对，电脑会认为这是两个完全不同的杯子！*
